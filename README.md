# 3-Line-News-Digest (3줄 뉴스 요약 번역기)

## 1. 프로젝트 개요
이 프로젝트는 영어로 된 긴 뉴스 기사를 입력받아, **3줄로 요약**하고 **한글로 번역**해주는 프로그램입니다.
OpenSource SW 텀 프로젝트 과제로 제작되었습니다.

## 2. 사용한 패키지 및 버전
* Python
* transformers
* torch
* sentencepiece
